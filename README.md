# Binary Classification with Bank Dataset

## Overview
This project solves a binary classification problem using the Bank Marketing dataset from Kaggle.

## Model Used
- XGBoost Classifier
- One-Hot Encoding for categorical features

## Evaluation Metric
- ROC-AUC Score (~0.96 on validation set)

## Files
- notebook5070cd1592.ipynb : Model training & evaluation
- submission.csv : Final Kaggle submission file

## Platform
- Kaggle Notebook
- Python, Pandas, Scikit-learn, XGBoost
